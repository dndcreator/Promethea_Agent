# Support

## Getting Help
- Open a GitHub issue for bugs/features.
- Use discussions/chat for usage questions.

## Before Asking
- Include exact command and error logs.
- Include environment (OS, Python version, commit).
- Confirm whether this is live mode (`PROMETHEA_LIVE_TEST=1`) or local unit mode.

## Response Scope
Community support is best effort.
For security issues, use `SECURITY.md` process.
