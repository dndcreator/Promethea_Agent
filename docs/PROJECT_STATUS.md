# Project Status

## Current stage

**Public Preview / Active Development**

Promethea is not presented as a finalized commercial release. It is a serious runtime project with usable core capabilities and active hardening.

## What is already usable

- Local runtime boot and exploration through UI/API/CLI
- Multi-user scoped session/config/memory paths
- Memory backend options and recall/write flow foundations
- Workflow runtime with checkpoint/resume behavior
- Policy/sandbox-aware tool execution paths
- Trace/audit visibility for runtime operations

## What is not finalized yet

- Broad, public benchmark coverage across multiple external suites
- Packaging/distribution polish for frictionless onboarding
- Full demo asset set (final video pack, polished graphics)
- Some docs and examples still being consolidated

## What we are validating next

- Stronger end-to-end benchmark reproducibility
- Reliability across longer workflows and mixed tool paths
- Better operator guidance for deployment/incident scenarios
- Tighter public onboarding loop from first visit to first successful run

## Who should follow the project now

- Agent infrastructure builders who care about runtime concerns beyond prompt wrappers
- Teams needing local-first control with multi-user boundaries
- Engineers evaluating memory/workflow/audit/policy as integrated runtime capabilities
- Contributors interested in practical agent systems, not only model prompting experiments
