# Roadmap

This roadmap focuses on product-critical open-source milestones.

## Now (v1 Hardening)

- Stabilize protocol-first runtime semantics and endpoint parity.
- Expand business-realistic test coverage (chat/tool/workflow/memory/voice journeys).
- Complete runtime observability consistency (trace/audit/readiness).
- Close configuration UX gaps (simple vs advanced config clarity).

## Next (v1.x Ecosystem)

- Official tool-pack maturity for general-purpose agent tasks.
- Better operator docs for deployment, backup, and incident response.
- CI improvements for broader platform verification.
- External developer integration examples (API + CLI).

## Later (v2 Direction)

- Deeper workflow orchestration capabilities for complex multi-step tasks.
- Stronger plugin/channel ecosystem and SDK experience.
- Richer governance automation around compatibility and deprecation.

## Non-goals (Current Cycle)

- Breaking protocol changes without explicit versioning.
- Feature growth that weakens user-boundary security.
